Note¶

how to write trace-event:

  1. 在 trace-event 新增 trace function 的 prototype 跟 printf format string.

  2. 在 source code 要 trace 處呼叫 trace function, 把要紀錄的資訊填入 trace function 的參數.

  3. source 中需要 #include "trace.h"

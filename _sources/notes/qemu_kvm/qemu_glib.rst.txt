QEMU GLib dependencies
======================

`GLib <https://developer.gnome.org/glib/stable/glib.html>`_ means Gnome GLib.
GLib is general-purpose utility library, which provides many useful data types, macros, type conversions, string utilities, file utilities, a mainloop abstraction, and so on.

- dependencies

  - reference: https://developer.gnome.org/glib/stable/
  - Basic Type: ``gpointer``. (GLib Fundamentals/Basic Types)
  - GLib Data Types: ``GList``, ``GHashTable`` . (GLib Data Types)
  - Event Loop: ``g_main_*``. (GLib Core Application Support/The Main Event Loop)
  - Memory Allocation: ``g_new``. (GLib Core Application Support/Memory Allocation)
